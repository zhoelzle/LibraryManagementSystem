# Library Management System

This repository contains an ASP.NET Core API project and a Vue.js frontend for a Library Management System.

## Getting Started

### API (ASP.NET Core)

1. **Opening in Visual Studio (2022)**:
   - Navigate to `API` folder.
   - Open `Library.sln` solution file in Visual Studio 2022.
   - Build and run the solution to start the ASP.NET Core API.

### UI (Vue.js)

1. **Opening in Visual Studio Code**:
   - Navigate to `UI` folder.
   - Open the folder in Visual Studio Code.
   - Install dependencies:
     ```bash
     npm install
     ```
   - Start the development server:
     ```bash
     npm run serve
     ```
   - Open your web browser and navigate to the provided URL (usually `http://localhost:8080`).

## Project Structure

- **API**: Contains the ASP.NET Core API for backend functionality.
- **UI**: Contains the Vue.js frontend for user interface.
